## Why Most Prompts Fail

Before the templates: here's what goes wrong.

**Common mistakes:**
- Too vague ("write something good")
- No format specified (you get whatever the AI feels like)
- Missing context (AI has to guess)
- No constraints (responses wander)

**The fix:** Structure + Specificity + Context

---

## Template 1: The Expert Advisor

Use when you need advice or analysis.

```
You are an expert [ROLE] with 20+ years of experience.

Context: [YOUR SITUATION]

I need you to:
1. [SPECIFIC ASK #1]
2. [SPECIFIC ASK #2]

Constraints:
- Be specific and actionable
- Include examples where helpful
- Flag any assumptions you're making

Format: [BULLET POINTS / NUMBERED LIST / PROSE]
```

**Example:**
```
You are an expert startup advisor with 20+ years of experience.

Context: I'm building an AI tool that analyzes prompts. I have a working product but zero customers.

I need you to:
1. Identify the 3 most effective customer acquisition channels for this type of product
2. Give me a specific action plan for the next 7 days

Constraints:
- Be specific and actionable
- Include examples where helpful
- Assume I have $0 marketing budget

Format: Numbered list with sub-bullets
```

## Template 2: The Content Creator

Use for writing any content.

```
Write a [CONTENT TYPE] about [TOPIC].

Target audience: [WHO]
Tone: [CASUAL / PROFESSIONAL / TECHNICAL / etc.]
Length: [WORD COUNT or SHORT/MEDIUM/LONG]

Key points to include:
- [POINT 1]
- [POINT 2]
- [POINT 3]

Avoid:
- [THING TO AVOID]

Format: [STRUCTURE - headers, bullets, etc.]
```

---

## Template 3: The Code Helper

Use for programming tasks.

```
I'm working in [LANGUAGE/FRAMEWORK].

Current code:
```[language]
[YOUR CODE]
```

I want to: [WHAT YOU'RE TRYING TO DO]

Problem: [WHAT'S NOT WORKING or WHAT YOU NEED]

Requirements:
- [REQUIREMENT 1]
- [REQUIREMENT 2]

Please provide:
1. The solution with explanation
2. Any edge cases to consider
```

---

## Template 4: The Analyzer

Use for breaking down complex topics.

```
Analyze [TOPIC/THING] from the perspective of [ANGLE].

I need to understand:
1. [QUESTION 1]
2. [QUESTION 2]
3. [QUESTION 3]

Context: [WHY YOU NEED THIS / YOUR BACKGROUND]

Output format:
- Start with a 2-sentence summary
- Then detailed breakdown
- End with actionable takeaways
```

---

## Template 5: The Brainstormer

Use for generating ideas.

```
Generate [NUMBER] ideas for [TOPIC/PROBLEM].

Context: [YOUR SITUATION]

Constraints:
- [CONSTRAINT 1 - e.g., "must be achievable in one week"]
- [CONSTRAINT 2 - e.g., "no budget required"]

For each idea, include:
- The idea in one sentence
- Why it might work
- First step to implement

Prioritize: [CREATIVITY / PRACTICALITY / BOTH]
```

## The Meta-Template

When you're not sure which template to use:

```
I need help with [TASK].

Context:
- My situation: [BRIEF BACKGROUND]
- My goal: [WHAT SUCCESS LOOKS LIKE]
- My constraints: [TIME/BUDGET/SKILLS/etc.]

Please:
1. [PRIMARY ASK]
2. [SECONDARY ASK if any]

Format your response as: [STRUCTURE]
```

## Pro Tips

1. **Be specific about format** ‚Äî "Give me 5 bullet points" beats "give me some ideas"

2. **Provide context** ‚Äî The more relevant background, the better the output

3. **Set constraints** ‚Äî Limits force creativity and relevance

4. **Ask for reasoning** ‚Äî Add "explain your thinking" to get better answers

5. **Iterate** ‚Äî First response not perfect? Say "good start, but make it more [X]"
